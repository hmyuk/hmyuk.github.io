## (1). sidebar navigation 수정



- 블로그 글 상단 설정에 author_profile을  false로 해줍니다.

```
author_profile: false
```



- _data > Navigation.yml파일을 아래와 같이 수정합니다.

```
docs:
  - title: "IT"
    children:
      - title : "Category"
        url : /categories/
      - title : "Tags"
        url : /tags/
```

- 블로그 글 상단 설정에 sidebar도 추가 해줍니다.

```
sidebar:
  nav:docs

```

- 설정이 완료되면 블로그에 들어가 navigation이 잘 나오는지 확인해 봅니다.
